# Feature Requests and Future Capabilities
1. Add the ability to import-as-new verses from a simple text file, with just the reference and the verse.
2. Future - Work with non-verse data that the user wants to memorize
3. Future - Have different methods to learn a verse: periodic review, practice/test, pictorial cue, audible cue, flash cards, read out loud, memory museum
4. Future - Have different ways to practice and test: Fill in the blank, type the first letter, choose-a-word, scramble words, scramble sections
5. Future - Integrate with AI to help with reviewing, practice/test, point to online resources
